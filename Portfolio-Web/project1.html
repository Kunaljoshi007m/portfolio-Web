Description

CLI Code Overview
The CLI portion of the code focuses on generating and managing government-related documents such as Aadhaar cards, PAN cards, and driving licenses. It uses SQLite for database management and handles input and output through the command line.

Initialization
Database Connection: Establishes a connection to an SQLite database named government_database.db. Creates tables if they don't exist:

  • Aadhaar: Stores Aadhaar card details.
  • pan_card: Stores PAN card application details.
  • driving_license: Stores driving license details.

Functions
generate_aadhaar_number(): Generates a random 12-digit Aadhaar number. Returns this number to be used in the Aadhaar details entry.
generate_pan_number(): Generates a random PAN card number. The format is four random uppercase letters, followed by four random digits, and ending with one random uppercase letter.
generate_license_number(): Generates a unique driving license number using a UUID and truncating it to 12 characters.
show_aadhaar_details(): Collects user input for Aadhaar card details:

  • Name
  • Phone number
  • Date of Birth (DOB)
  • Address
  • Gender
Validates the input (ensures all fields are filled and name format is correct). Inserts the details into the Aadhaar table in the database. Displays the Aadhaar number to the user.
apply_for_pan_card(): Collects user input for PAN card application:
  • Name
  • Phone number
  • Date of Birth (DOB)
  • Address
  • Gender
Validates the input (ensures all fields are filled and name format is correct). Generates a PAN number and inserts the details into the Pan Card table in the database. Displays the PAN number to the user.
apply_for_driving_license(): Collects user input for driving license application:
  • Name
  • Date of Birth (DOB)
  • Address
  • Phone number
  • Gender
Validates the input (ensures all fields are filled and name format is correct). Generates a driving license number and inserts the details into the Driving License table in the database. Displays the driving license number to the user.

Database Management
Tables:

  • Aadhaar: Columns for name, phone_number, dob, address, gender, aadhaar_number.
  • pan_card: Columns for name, phone_number, dob, address, gender, pan_card_number.
  • driving_license: Columns for name, phone_number, dob, address, gender, license_number.
Database Operations:
  • Creates tables if they don't already exist.
  • Inserts records into respective tables based on user input.

Error Handling
Handles errors related to database operations and user input: Displays appropriate error messages if input validation fails or if there are issues with database operations.

Additional Notes
CLI Output: Outputs information directly to the command line, including success messages and generated numbers.
No GUI: All interactions are done through command-line prompts rather than graphical interfaces.
